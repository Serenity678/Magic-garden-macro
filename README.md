# Magic-garden-macro
This is a small-scale macro project with move logic intended for use with the Discord game "Magic Garden." 

There are two files in this release:
Magic Garden Script
Magic Garden Test

Magic Garden Script is the main project of this repo and will automate farming and moving around a single 10x10 plane without player input. 

Magic Garden Test is a program run by the user to check specific values and get information about their environment in case they want to make adjustments to the program. It is not a debugging tool. 

Have fun!
